# vue-my-todo
Веб-приложение vue-my-todo  А также позволит добавлять новые и
управлять информацией о них.
Технологический стек
Vue + VueX + vue-router + Babel + Sass + ESLint+ Element UI
## Установка проекта
```
yarn install
```

### Запуск
```
yarn run serve
```

### Собрать
```
yarn run build
```

### Тестирование
```
yarn run test
```

### Для исправления
```
yarn run lint
```
