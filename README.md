# vue-my-todo
Веб-приложение vue-my-todo  В общем есть список задач, задачи можно удалять, добавлять.
Технологический стек
Vue + Babel + Sass + ESLint+ Element UI
## Установка проекта
```
yarn install
```

### Запуск
```
yarn run serve
```

### Собрать
```
yarn run build
```

### Тестирование
```
yarn run test
```

### Для исправления
```
yarn run lint
```
